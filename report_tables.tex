\section*{Statistical Analysis Details}
The comparison between GP and MLP models was conducted using the Wilcoxon signed-rank test with significance level $a = 0.05$. This non-parametric test was chosen due to the small sample size (3 seeds) and no assumption of normal distribution. A p-value $< 0.05$ would indicate a statistically significant difference between the models.
